# MUGGUS ROYALE
Ett helt otroligt quizspel. 

## Android

Follow the [Environment Setup Guide](https://reactnative.dev/docs/environment-setup).

```
npm run android
```

## iOS

Install XCode, the CLI tools when prompted and [CocoaPods](https://cocoapods.org). 

```
cd ios
pod install
cd ..
npm run ios
```
